Work Description:

Attendance manager is a management tool for students and admin of a college's system. The student can login at their lecture times and the entry is inserted in a database using PHP and JS. The project is part of academic project during Semester 4 for subject Software Engineering.

The website design has the following features:

  • Attendance Login -

    • The students enter their college ID to add their entry in college's database.

  • Records -

    • Displays full database table.
