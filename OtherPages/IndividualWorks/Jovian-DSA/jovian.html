Work Description:

The project is Data Structure and Algorithm analysis of a problem on GeeksforGeeks.

Link to the problem:
https://www.geeksforgeeks.org/maximum-profit-by-buying-and-selling-a-share-at-most-twice/

Problem Statement:
Calculate the maximum profit that can be generated from share prices in a day with atmost two transactions. The project analyses the method used for solving the problem and calculating the time and space complexity of the program.
