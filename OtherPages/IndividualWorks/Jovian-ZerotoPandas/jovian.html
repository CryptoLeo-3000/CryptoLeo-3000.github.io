Work Description:

This project focuses on analysis of a kaggle dataset on renewable energy. It includes processes like data prep and cleaning, looking for null values and removing and them, analysing the dataset with visualizations and graphs and answering various questions from the dataset.

The dataset has been dowloaded from:
https://bit.ly/321xFn8

The project uses python libraries:

  • Opendatasets
  • Pandas
  • Numpy
  • Matplotlib
  • Calmap
  • Seaborn
  • Plotly
And their various functions and methods.
